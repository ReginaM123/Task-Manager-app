 Task-Manager-app
A simple and responsive task manager application built with HTML, CSS, and JavaScript. It allows users to add and delete tasks, with data stored in the browser using localStorage. This project demonstrates DOM manipulation, basic state management, and clean front-end structure.

Task Manager App

A simple task manager built with HTML, CSS, and JavaScript.

 Features
- Add tasks
- Delete tasks
- Data stored in localStorage
- Responsive and clean UI

 Technologies Used
- HTML
- CSS
- JavaScript

 How to Run
Just open `index.html` in your browser.
